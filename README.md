# JavaScript Learning Journey 🚀

A structured repo documenting my journey learning JavaScript from the basics through the DOM .

## 📂 Folder Structure

| Folder | Topic |
|---|---|
| `01_basics` | JavaScript fundamentals — variables, data types, operators |
| `02_basics` | Core concepts continued — functions, scope |
| `03_basics` | IIFEs and related patterns |
| `04_control_flow` | Conditionals, loops, control flow logic |
| `05_iterations` | Advanced iteration techniques |
| `06_DOM` | DOM manipulation — selecting, modifying, and styling elements |

## ▶️ How to Run

Most files are plain HTML/JS. To view them:

1. Open the relevant `.html` file in your browser, or
2. Use the **Live Server** extension in VS Code:
   - Right-click the `.html` file → **Open with Live Server**
